Connect Four

Python, OO Design, MVC, Processing GUI framework, JavaScript

Implemented a Processing GUI framework based "Connect Four" game with MVC architecture. This game supported both human-human and human-computer Playing. Multiple components includes:

1) Game rendering component.

2) Gaming logic handling controller.

3) Score recording and racking system.

4) AI robot based on minimax algorithm with alpha-beta pruning.

5) Rewrite the python code to p5.js , and deploy the web-based playable version on website.
